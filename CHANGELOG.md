# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-05-31
### Added
- Native ES Module (ESM) support alongside CommonJS
- Added `tsup` for dual-format compilation
- Configured Node.js engines constraint (>= 18)
- Added `LICENSE` (MIT)
- Initial Changelog tracking

## [1.0.0] - 2026-05-31
### Added
- Initial release of the `swiftsmsgh-api-sdk` package
- Support for SMS, Voice, MMS, OTP, WhatsApp, and Viber
- Complete typescript definitions and automated documentation
- Campaign and Contacts management APIs
